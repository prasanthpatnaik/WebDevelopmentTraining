# Rock Paper Scissors

In this project, weare going to build the `Rock Paper Scissors` game using HTML, CSS and JS.

### Rules -

This game has 2 players and each player can pick one of the following 3 choices-

- Rock
- Paper
- Scissors

Both the players compare their pick with each other and whoever has the greater pick wins.

The order of the choices is as follows-
`Rock > Scissors`
`Scissors > Paper`
`Paper > Rock`

##### Example-

When Player1 chooses Rock and Player2 chooses Scissors, `Rock beats Scissors` and Player1 wins.
When Player1 chooses Paper and Player2 chooses Scissors, `Scissors cuts Paper` and Player2 wins.
When Player1 chooses Rock and Player2 chooses Paper, `Paper wraps Rock` and Player2 wins.
